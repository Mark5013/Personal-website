Projects

PomoTracker.tech

Full stack productivity web app based off the pomodoro method. Implemented a REST api for seamless end to end user experience. Users can create/edit/delete/drag any task they wish. Users stats will be tracked and they have the option of viewing their stats on their profile page. Timer lengths can also be configured in the settings page.

Pokemon TCG Search

Website that allows users to search for Pokemon TCG cards. This was made using the pokemon TCG api to query for pokemon card information. EJS templating for easy page rendering, node.js/express on the backend and js/bootstrap for the frontend. Users can also sign up/in via google oauth and save cards to their collection for easy access. Cookies are used so users aren't logged out just by closing out of the tab. Implemented with mongodDB/mongoose/passport and other passport packages. Currently hosted on heroku.

Todo List

Browser based todo list app that lets users make a todo list. Also allows users to create a todo list of any topic by using dynamic routing with express. Also used mongoDB/mongoose to store user data. Currently hosted on heroku.

Note taking

Note taking app that was written using React. Users can add/delete notes. Performs validation checks on user input to ensure that text is entered in both the title and body section. Uses hooks like useReducer, useState, and useContext.

Discord weather bot

Simple discord bot that I made using the weather.js api to query for weather information based off of the location inputed. Other commands were also implemented using discordjs such as retreiving a users profile image. This was made using js/node.js.
